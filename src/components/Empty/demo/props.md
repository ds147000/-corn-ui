# Empty 空状态
> 空状态时的展示占位图。


### 什么时候使用
 - 当目前没有数据时，用于显式的用户提示。
 - 初始化场景时的引导创建流程。


<demo>


## Props
| 属性 | 必填 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- | --- |
| src | 否 | 自定义缺省图片地址 | `string` | `Empty.PRESENTED_IMAGE_DEFAULT` |
| size | 否 |  大小 | `large` `middle` `small`  | `middle` |
| text | 否 |  自定义描述内容	 | `string` |  |
| wrapperTop | 否 |  顶部间距 | `number` | `30px` |
| wrapperBottom | 否 |  底部间距 | `number` | `30px` |
